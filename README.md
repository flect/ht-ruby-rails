ht-ruby-rails
=============

Sample project that use Ruby on Rails(4.0.0) for Heroku training

How to deploy
--------------

### Step1  

Clone app from Github  

```
$ git clone git@github.com:flect/ht-ruby-rails.git
$ cd ht-ruby-rails/
```

### Step2  

Create Heroku application

```
$ heroku create
```

### Step3  

Deploy application
```
$ git push heroku master
```

### Step 4

Visit application

```
$ heroku open
```
